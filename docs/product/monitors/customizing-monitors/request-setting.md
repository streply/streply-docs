# Request setting

In request setting you can change:

- Request method
  - GET
  - POST
  - PUT
  - PATCH
  - HEAD
- Request timeout
  - 1m
  - 45s
  - 30s
  - 15s
  - 10s
  - 5s
  - 3s
  - 2s
- Request body (for POST, PUT and PATCH methods)
- Follow redirects
- Custom header
